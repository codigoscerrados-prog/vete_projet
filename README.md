# Veterinaria (Angular + localStorage)

Este repositorio contiene el proyecto **Veterinaria** desarrollado en Angular con TypeScript.

## Estructura

- `veterinaria/`: aplicacion web principal.
- `Regla de oro.docx`: documento original del curso.

## Inicio rapido

```bash
cd veterinaria
npm install
npm start
```

Abrir `http://localhost:4200/`.

## Nota

El proyecto no usa backend ni base de datos SQL. Toda la persistencia se hace con `localStorage`.
